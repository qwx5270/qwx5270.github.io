Markdown语法

标题

markdown标题有两种格式

  • 使用 = 和 - 作为一级标题和二级标题 如下:

    这展示的是一级标题
    ================

    这展示的是二级标题
    ---------------------------

  • 使用 # 号作为标记

    # h1
    ## h2
    ### h3
    #### h4
    ##### h5
    ###### h6

    字体强调

    *斜体* 和 _斜体_

    *加粗* 和 __加粗__

    ***粗斜体文本*** 和 ___粗斜体文本___

    ~删除线 和 <u>下划线</u>

    [^脚注(注明的文本)]

    创建脚注格式类似这样 ^haha

分割线

在一行中用三个及以上的星号、减号、底线来建立一个分隔线, 行内不能有其他字符,空格符除外

*** 和 --- 和 ___

列表

* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

1. 第一项
2. 第二项
3. 第三项

列表嵌套

子列表中的选项添加四个空格

  1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
  2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第一个元素

块级元素

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号

区块中使用列表

> ## This is a header.
>
> 1. This is the first list item.
> 2. This is the second list item.
>
> Here's some example code:
>
> return shell_exec("echo $input | $markdown_script");

This is a header.

  1. This is the first list item.
  2. This is the second list item.

Here’s some example code:

return shell_exec("echo $input | $markdown_script");

列表中使用区块

要在列表项目内放进区块，那么就需要在 > 前添加四个空格的缩进
* 第一项
  > This is the first level of quoting.
  > Back to the first level.
* 第二项

###代码区块

  1. 代码块: 缩进 4 个空格或是 1 个制表符
    <?php
    echo ‘RUNOOB’;
    function () {

    echo 'test';

    }

  2. 行内式：如果在一个行内需要引用代码, 只要用反引号`引起来就好
    printf('hello, world')

  3. 多行代码块与语法高亮：在需要高亮的代码块的前一行及后一行使用三个单反引号“`”包裹

链接

[链接名称](链接地址)
[baidu](www.baidu.com)

这是一个链接—> 我的博客

####高级链接
链接也可以用变量来代替，文档末尾附带变量地址

I get 10 times more traffic from [Google] 1 than from
[Yahoo] 2 or [MSN] 3.

图片

图片的语法格式以下

![alt 属性文本](图片地址)

![alt 属性文本](图片地址 "可选标题")

表格

表头一 表头二 表头三
项目一 项目二 项目三
项目2一 项目2二 项目2三

注：三个短斜杠左右的冒号用于控制对齐方式，只放置左边冒号表示文字居左，只放置右边冒号表示文字居右，如果两边都放置冒号表示文字居中。

其他的高级技巧

目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等

使用 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Del</kbd> 重启电脑

使用 Ctrl+Alt+Del 重启电脑

反斜杠转义
自动链接

Markdown 支持一种 “自动” 创建 URL 和 email 地址链接的简短形式: 只需用尖括号包围 URL 或 email 地址即可 

<qwx5270@gmail.com>      
<https://qwx5270.github.io/>
